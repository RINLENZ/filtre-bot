# FiltreBot Telegram

Bot Telegram intelligent de filtrage de fichiers par mot-clé dans plusieurs canaux.

## 📦 Installation

```bash
pip install -r requirements.txt
```

## 🚀 Lancement

```bash
python3 filtre_bot.py
```

## 🔁 Relancer automatiquement (Linux VPS)

```bash
chmod +x start.sh
./start.sh
```
